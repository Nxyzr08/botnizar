## Installation

Pastiin udah install nodejs ya

**Clone Repository**

```bash
git clone https://github.com/Nxyzr08/botnizar
```
```bash
cd botnizar
```

**Install Library**

```bash
npm i
```

**Set Up Environment Variable**

Buat file `.env` terus tambahin :

```env
TELEGRAM_TOKEN = YOUR_TELEGRAM_TOKEN
```

**Run Bot**

```bash
npm run bot
```
